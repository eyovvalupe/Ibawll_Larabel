## Install

    git clone git@github.com:arestivo/lbaw-laravel.git
    cd lbaw-laravel/
    sudo chown -R www-data:www-data web/todo/storage/
    cd web/todo
    composer install

## Start    

From project root:    
    
    docker-compose up

## Testing

Navigate to:

    http://localhost:8080/list
